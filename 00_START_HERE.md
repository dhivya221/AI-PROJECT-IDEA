## 🚀 How to Get Started (3 Simple Steps)

### Step 1: Get API Keys (5 minutes)
```bash
OpenAI:     https://platform.openai.com/api-keys
Pinecone:   https://www.pinecone.io (create serverless index)
```

### Step 2: Run Backend (2 minutes)
```bash
cd backend
python -m venv venv
venv\Scripts\activate  # Windows
pip install -r requirements.txt
cp .env.example .env   # Add your API keys
python main.py
```

### Step 3: Run Frontend (1 minute)
```bash
cd frontend
npm install
npm start
```

**That's it! Visit http://localhost:3001** 🎉

## 📋 What Each Component Does

### Domain Analyzer
Understands what you're asking about by analyzing:
- Market context and size
- Stakeholders and competitors
- Technical requirements
- Industry trends

### Idea Generator
Creates 5 innovative project ideas with:
- Creative titles and descriptions
- Use cases for each idea
- Potential market impact
- Implementation approaches

### Complexity Estimator
Predicts development effort:
- Complexity score (0-10)
- Development timeline
- Difficulty level
- Key technical challenges

### Tech Stack Recommender
Suggests optimal tools:
- Backend framework
- Frontend framework
- Database solution
- Deployment platform
- AI/ML libraries
- Rationale for each choice

### Scoring Module
Evaluates ideas on:
- Feasibility (0-100)
- Innovation (0-100)
- Market Potential (0-100)
- Composite Score
- Detailed reasoning

### Recommendation Engine
Provides strategic guidance:
- Priority ranking
- Team composition
- Partnership needs
- Risk mitigation
- Phased roadmap

### Vector Search/RAG
Maintains idea memory:
- Converts ideas to embeddings
- Stores in Pinecone DB
- Enables semantic search
- Provides contextual retrieval

### MCP Server
Exposes all tools:
- Model Context Protocol
- Claude integration
- AI-to-AI collaboration
- Standardized interface
